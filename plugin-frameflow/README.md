# Frameflow Framer Plugin

This plugin is the Framer-side bridge for the local Frameflow pipeline.

## Purpose
- Collect URL or screenshot references inside Framer
- Save build instructions for the local generator
- Later insert generated sections, assets, and code components back into Framer

## Dev
1. Open this folder as a Framer plugin project
2. Run `npm install`
3. Run `npm run dev`
4. In Framer, enable Developer Tools and open the development plugin
